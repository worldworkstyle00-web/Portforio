📋 仕事の進め方

在宅でのご依頼でも、不安なく進めていただけるよう、仕事の進め方をできるだけシンプルに、わかりやすく整えています。

1. ご相談・ヒアリング
まずは状況とご希望を整理するところから。
  • どのような課題があるか
  • 何を実現したいか
  • いつ頃までに必要か
  • 既にある資料・画面・サイトなどの有無

必要に応じて、Zoom / Google Meet 等で画面共有しながらのヒアリングも可能です。

2. 現状の整理とご提案
「すぐできること」と「段階を分けるべきこと」を切り分けます。
  • すぐに着手できる範囲の整理
  • 段階を分けた方が良い部分の洗い出し
  • お手伝いできること／できないことの明確化

作業内容の概要や、スケジュール感をテキストで共有し、ご相談しながら決めていきます。

3. 方針の確定
進め方を事前にすり合わせてから、実作業に入ります。
  • 使用するツール（共有ドキュメント・チャットツールなど）の確認
  • 連絡頻度・中間レビューのタイミングの決定
  • 成果物の形式や、確認フローのすり合わせ
4. 作業の実施
在宅で、合意した内容に沿って進めます。
  • 手順書・マニュアルの草案作成
  • Webページの構成案・ワイヤーフレーム作成
  • HTML / CSS / JavaScript を用いた実装
  • 社内向け案内文・ドキュメントの整理
  • IT資産管理・棚卸し関連資料の整備　など

重要な判断が必要な場合は、その都度ご相談のうえで進めます。

5. 中間レビュー
早い段階で認識のズレをなくします。
  • 中間成果物の共有
  • 表現や構成に関するご意見・ご要望の確認
  • 追加で必要な情報や修正点の洗い出し

フィードバック内容を踏まえて、最終版に向けて調整します。

6. 納品・最終確認
実際の運用をイメージしながら、最終チェックを行います。
  • 文言・レイアウト・構成の最終確認
  • 運用上の問題がないかどうかの確認
  • 軽微な修正への対応
🔁 7. アフターフォロー（任意）

実際に運用してみてから見えてくる「使いにくさ」や「説明の不足」にも対応します。

運用の中で、

  • 使いにくい部分が見つかった
  • 説明を追加したい箇所が出てきた
  • ページやドキュメントを更新したい

といった場合には、継続してご相談いただけます。
※運用やアフターフォローを含めたご契約も可能です。
※納品後に継続的な運用やフォローが必要になった場合は、別途お見積もりいたします。
