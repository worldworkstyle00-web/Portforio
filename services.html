🧾 提供できる業務

在宅で完結できる業務を中心に、これまでの経験とスキルを組み合わせて対応しています。
依頼内容の背景や目的を伺い、現実的で実効性のある形に落とし込んでいきます。

1. 手順書・マニュアル・ドキュメント作成
「理解してから作業できる」資料づくりを重視しています。
  • PCキッティング手順書の作成・改訂
  • 社内システムの利用手順書（図解を含む説明資料など）
  • IT資産管理・棚卸しの運用ルール整理
  • 社内向け案内ページ・フロー説明の文章整備
  • 情報量の多い内容を、章立て・図解で整理し直す
2. Webページ制作（小規模・静的サイト）
小さく始めたい方向けの、シンプルなWebページ制作。
  • HTML / CSS / JavaScript を用いた静的ページ
  • ポートフォリオ・サービス紹介などの小規模サイト
  • スマートフォン対応のレスポンシブデザイン
  • シンプルなToDoアプリなど、軽量なフロント実装
  • 既存ページの文言修正や構成の見直し
3. ライティング・文章リライト
読み手が迷わず理解できる文章を意識しています。
  • 商品説明・サービス紹介文の作成
  • ターゲットに合わせたトーン調整（主婦層向けなど）
  • 情報過多な文章の整理・リライト
  • Webページ・社内案内文の読みやすさ改善
4. コンサル業務（在宅・継続的サポート）
「何から手を付ければいいか」を一緒に整理していきます。
  • 業務や現状の整理・課題の可視化
  • 整備すべき箇所の優先順位付け
  • 運用ルールや手順の再構成に関する相談
  • 改善に向けたステップ設計・伴走支援
  • 在宅前提での長期的なフォロー
5. 社内IT運用・情報システムまわりの支援
情報システム部門での経験をベースにしたバックオフィス支援。
  • IT資産ライフサイクル管理の運用整理
  • 在庫機器・返却機器の管理表・台帳整備
  • アカウント・権限整理フローの文書化
  • 棚卸しに向けた資料作成・整合性確認の支援
  • 管理DBの整理ルール・運用ドキュメント作成
6. DTP・簡易デザイン
「伝わるレイアウト」を重視した印刷物制作。
  • 名刺・ハガキ・チラシ・簡易パンフレットのデータ作成
  • Illustrator / Photoshop を用いたレイアウト調整
  • 印刷前提のデータ確認・入稿データの整え
